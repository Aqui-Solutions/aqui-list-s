# Aqui List (S)
 Aqui's lightweight, simple base version for Aqui List. "S" stands for "small".
 # Built with:
- JavaScript;
- React Native (+ Expo).

 # It supports:
- Single list of items;
- Item deleting, editing
- Unlimited item creation
